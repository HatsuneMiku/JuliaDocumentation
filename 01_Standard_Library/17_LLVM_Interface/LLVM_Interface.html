LLVM Interface
LLVMインターフェース LLVM とは、コンパイル時、リンク時、実行時などあらゆる時点でプログラムを最適化するよう設計された、任意のプログラミング言語に対応可能なコンパイラ基盤で ある。 当初は、LLVM の名称の由来は、Low Level Virtual Machine (低水準仮想機械) の略で .. 現在は、何の頭文字でもないとしている。

llvmcall(IR::String, ReturnType, (ArgumentType1, ...), ArgumentValue1, ...)     Call LLVM IR string in the first argument. Similar to an LLVM function define block, arguments are available as consecutive unnamed SSA variables (%0, %1, etc.).
    Note that the argument type tuple must be a literal tuple, and not a tuple-valued variable or expression.
    Each ArgumentValue to llvmcall will be converted to the corresponding ArgumentType, by automatic insertion of calls to unsafe_convert(ArgumentType, cconvert(ArgumentType, ArgumentValue)). (see also the documentation for each of these functions for further details). In most cases, this simply results in a call to convert(ArgumentType, ArgumentValue).
    See test/llvmcall.jl for usage examples.
