数学記号＆機 能の拡張ドキュメントはこちらです。

symbol meaning
@m マクロメー トルを呼び出します。スペースで区切られた表現が続きます
! 否定演算子
a!( ) 関数名の末 尾！関数は、引数を修正することを示しています
# 1行コメントを開始
#= 複数行のコ メントを（これらはネスト可能です）開始
=# 複数行のコメントを終わらせます
$ ビットごとのXOR演算、文字 列と表現補間
% 剰余演算子
^ 指数演算子
& ビット論理積
&& かつ
|
ビット論理和
|| または
* 乗算、または行列乗算
() 空のタプル
~ ビット単位のないオペレータ
\ バックスラッシュ演算子
' 複雑な転置演算子AH
a[] 配列インデックス
[,] 垂直連結
[;] また、垂直方向の連結
[  ] スペースで区切られた式と、水平方向の連 結
T{ } パラメトリックタイプのインスタンス化
{  } セル・配列構築（任意の 賛成でバージョン0.4で非推奨にな りました）
; 文の区切り（表示しない役割もある）
, 関数の引数やタプルコンポーネントの区切り
? 3項演算子 （条件付きtrueの場合：？falseの場合）
"" 文字列リテラルを区切ります
'' 区切り文字リテラル
` ` 外部プロセ ス（コマンド）の仕様を 区切ります
... 関数呼び出 しに引数をスプライスまたは可 変引数関数や型を宣言
. モジュール内 のオブジェクトや名前でのアクセスという名前の フィールドは、また、要素単 位の演算子を接頭辞
a:b 範囲 a, a+1, a+2, ..., b
a:s:b 範囲 a, a+s, a+2s, ..., b
: インデックス全体の寸法（1：終 了）
:: コンテキストに応じて、注釈を入力
:( ) 引用符で囲まれた式
:a シンボル a
